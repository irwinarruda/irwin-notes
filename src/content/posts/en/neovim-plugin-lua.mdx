## Why Lua?

Vimscript served its purpose for decades, but Lua brings a modern scripting workflow to Neovim.

- **Performance** - LuaJIT is much faster for computation-heavy tasks
- **Familiar syntax** - if you use JavaScript or Python, Lua feels natural quickly
- **First-class API** - `vim.api`, `vim.fn`, and `vim.opt` cover nearly everything
- **Ecosystem alignment** - popular plugins like telescope and treesitter are Lua-first

## Project Structure

The folder structure is predictable and easy to scale:

```text title="my-plugin.nvim"
my-plugin.nvim/
|- lua/
|  |- my-plugin/
|  |  |- init.lua
|  |  |- config.lua
|  |  '- core.lua
|- plugin/
|  '- my-plugin.lua
|- doc/
|  '- my-plugin.txt
'- README.md
```

`lua/` contains your modules. `plugin/` is auto-loaded by Neovim and is a good place for commands and startup wiring.

## Writing the Core

This example tracks line-count deltas after saves:

```lua title="lua/my-plugin/core.lua" {11-19}
local M = {}

local state = {}

function M.on_buf_write(bufnr)
  local lines = vim.api.nvim_buf_line_count(bufnr)
  local prev = state[bufnr]

  if prev and lines ~= prev then
    local delta = lines - prev
    local sign = delta > 0 and "+" or ""
    vim.notify(
      string.format("%s%d lines (%d total)", sign, delta, lines),
      vim.log.levels.INFO
    )
  end

  state[bufnr] = lines
end

function M.setup()
  vim.api.nvim_create_autocmd("BufWritePost", {
    callback = function(ev)
      M.on_buf_write(ev.buf)
    end,
  })
end

return M
```

## Registering Commands

Users expect discoverable commands:

```lua title="plugin/my-plugin.lua"
if vim.g.loaded_my_plugin then
  return
end

vim.g.loaded_my_plugin = true

vim.api.nvim_create_user_command("MyPluginEnable", function()
  require("my-plugin.core").setup()
  vim.notify("my-plugin enabled", vim.log.levels.INFO)
end, { desc = "Enable my-plugin line tracking" })
```

## Wrapping Up

The core loop is simple: structure modules, attach to editor events, and expose commands. Once that foundation is in place, Lua plugins compose naturally and stay maintainable.
